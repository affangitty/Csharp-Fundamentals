# Task 1: Factorial Calculator

## Objective
Create a console application that calculates the factorial of a number.

## Features
- Takes user input
- Validates input (positive integer)
- Uses loop to compute factorial
- Displays result

## Concepts Used
- Data types (int, long)
- Input/output (Console)
- Conditional statements
- Loops
- Methods

## How to Run
Compile and run Program.cs
