# Task 5: File I/O and Exception Handling

## Objective
Read data from a file, process it, and write results to another file.

## Features
- Reads input.txt
- Counts lines and words
- Writes results to output.txt
- Handles file-related exceptions

## Concepts Used
- File I/O (File.ReadAllLines, File.WriteAllText)
- Exception handling (try-catch)
- String processing

## How to Run
1. Place input.txt in same folder
2. Run: dotnet run
